Line Height (Altura da linha)

O Bootstrap oferece classes para controlar a altura da linha, que é a distância entre as linhas de texto. Isso pode melhorar a legibilidade e o espaçamento do conteúdo.

Font Size (Tamanhos Utilitários)

O Bootstrap inclui classes para controlar o tamanho de fonte, permitindo que você ajuste rapidamente o tamanho do texto sem escrever CSS personalizado. Essas classes são úteis para criar hierarquia visual e destacar partes específicas do conteúdo.
